Expense Tracker

A simple expense tracker application that enabled user to record their daily transaction locally. User have ability to add item list and put the transaction into category. This application build using SwiftUI with CoreData as persistance database. Transaction are grouped by date using fetch request section and can be filter by month. In the future it will be update with SwiftData and enable user to export monthly expense into editable document.
